# Orc Onomastics

**Author anchor:** Guttural consonant clusters, glottal apostrophes, -obbi/-oxi tribal suffixes  
**Source pool (frozen map names — do not copy verbatim):** Glivornaxi, Hedroscobbi, Xi'ivrach, Shazuihni, Uihonaii, Griebi, Tytheri, Thrygun, Tarni, Enti, Bloodreavers — place names: Grand Tolkarsus, Plenjorn Swamp, Bloodwood, Jeh Bli, Yugurbas Dyos, White Hills, Agony Stones

## Sound palette
- Consonants that recur: heavy clusters (gl/vr/dr/shr/thr/ry), guttural g/k/x, the sh/zh voiced fricatives; terminal nasal clusters (-ni, -xi, -bi)
- Vowel behavior: vowels are carriers between consonant masses, not the expressive element; i/u/a dominate; long vowel pairs allowed when both are i or u (Uihonaii = ui-ho-na-ii pattern — double-i terminal)
- Glottal apostrophe: marks a hard stop within a name (Xi'ivrach); use at most once per name; never word-initial or word-final
- Forbidden / off-style: soft l/w/h alone (reads halfling), flowing diphthongs (reads elven), -ling suffix (reads halfling/goblin), any name under 3 consonants total

## Morphemes & affixes
- Prefixes: Gli-/Hedr-/Xi-/Sha-/Uih- — consonant-mass openers; no semantic family; phoneme-cluster markers
- Suffixes (tribal, high-signal): **-obbi** (Hedroscobbi) and **-oxi/-naxi** (Glivornaxi) are the primary tribal-identity suffixes; a name ending in -obbi or -oxi reads as a tribal or clan identifier
- Suffixes (personal): -rach (Rachtype: Xi'ivrach), -gun (Thrygun), -ni/-ri (Shazuihni, Tytheri, Tarni) — personal name terminals
- Compounding: no compound logic; names are monolithic consonant masses, not root + suffix in a readable way

## Naming customs
- **Tribal / clan names** end in -obbi or -oxi/-naxi (Hedroscobbi, Glivornaxi); this is the strongest clan-identity signal in orc onomastics — always use it for a tribe or people-group name
- **Personal names** use the shorter terminals (-rach, -gun, -ni, -ri) or a truncated consonant-mass form (Tarni, Enti, Griebi)
- **Place names** split: orc-named places use consonant-mass roots (Grand Tolkarsus, Yugurbas Dyos); outsider-named places use English-descriptive (Bloodwood, Agony Stones, White Hills)
- **Apostrophe** appears in personal names only — not in place names; it signals identity/power within a name
- Glottal apostrophe distinguishes orc from goblin: goblins use reduplication; orcs use the glottal stop mid-mass

## Worked examples (derived, not copied)
- **Vraxunobbi** — tribal: Vrax- (vr+ax cluster) + -un- + -obbi (tribal terminal); a clan name
- **Ghrythan** — personal: Ghr- (voiced variant of thr-, cf. Thrygun) + -ythan (compressed -thr inversion); a warrior name
- **Kazixoxi** — tribal: Kaz- + -ix- + -oxi (tribal terminal); an alternate clan form
