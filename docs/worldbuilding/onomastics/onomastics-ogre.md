# Ogre Onomastics

> **sparse sample — extrapolate conservatively**

**Author anchor:** Two sub-registers — Sil- plains (territorial prefix) and Gro- swamp (cluster opener + oy diphthong); z-sound shared across both  
**Source pool (frozen — do not copy verbatim):** Silzar (Plains Ogre subrace), Groyza (Swamp Ogre subrace), Silwyn (plains territory)

## Sound palette
- **Silzar**: Sil + zar — Sil- opener (s+il), z interior, -ar terminal
- **Groyza**: Groi + za — Gr- cluster opener, -oy- diphthong, -za terminal
- **Silwyn**: Sil + wyn — Sil- opener (matching Silzar), -wyn terminal (w+yn nasal close)
- Z-sound is the shared marker: -zar (Silzar), -za (Groyza) — every confirmed name contains it
- Two distinct sub-registers corresponding to the two subraces:
  1. **Plains / Silzar**: Sil- prefix, z-interior, -ar/-wyn terminals
  2. **Swamp / Groyza**: Gr- cluster opener, -oy- diphthong, -za terminal
- Forbidden / off-style: z-free names (off-ogre); soft liquid openers without a hard consonant; names that lack z entirely

## Morphemes & affixes
- **Sil-** prefix: shared by Silzar (subrace) and Silwyn (territory) — marks plains/nomadic lineage; likely a territorial marker absorbed into personal naming
- **Z-sound**: universal — -zar (Silzar), -za (Groyza); the one phoneme both registers share; mandatory
- **-ar/-wyn terminals**: plains register (Silzar, Silwyn)
- **Gr- opener + -oy- diphthong + -za terminal**: swamp register (Groyza); the oy diphthong is darker and not used in plains names
- Sub-register mixing: possible in mixed-lineage or border-territory ogres, but each pure sub-register should stay internally consistent

## Naming customs
- The Sil- prefix appearing in both the subrace name and the territory name (Silwyn) suggests plains ogres use it as a shared clan-territorial marker — naming into the land
- Z-sound is non-negotiable; a name without it does not read as ogre in this corpus
- Swamp ogres (Groyza) favor the darker oy diphthong interior; plains ogres (Silzar/Silwyn) avoid it
- Place names follow the same register as their inhabitant subrace: plains territories use Sil- + nasal or w-close; swamp territories use Gr- + oy + z-close
- These are a two-gender, separately-tribing culture; naming conventions likely differ by gender but the phonemic anchors (Sil- prefix, z-sound) appear cross-gender
