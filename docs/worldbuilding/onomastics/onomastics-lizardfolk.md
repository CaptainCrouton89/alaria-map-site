# Lizardfolk Onomastics

> **sparse sample — extrapolate conservatively**

**Author anchor:** Ix- prefix invariant with complex mid-body consonants  
**Source pool (frozen map names — do not copy verbatim):** Ixlorett, Ixmeglyakuk, Ixtyrann, Ixvasyla

## Sound palette
- Consonants that recur: hard initial Ix- (short vowel + x), complex mid-body clusters (mgl/tyr/vas), terminal -tt/-uk/-nn/-la
- Vowel behavior: short vowel between consonant clusters; mid-word vowels (e, a, y) act as separators, not expressive elements
- Forbidden / off-style: any name that does NOT begin Ix-; soft flowing vowel chains; -ling/-heim endings

## Morphemes & affixes
- **Prefix (invariant): Ix-** — every attested lizardfolk name begins Ix-; this is the single strongest rule in this pool; treat it as obligatory for lizardfolk names
- Mid-body: consonant cluster + short vowel + terminal (mgl + ya + kuk; tyr + ann; vas + yla; lor + ett)
- Terminals: -ett/-uk/-ann/-yla — no clear semantic family; phoneme-terminal markers

## Naming customs
- The Ix- prefix is the signal — it immediately identifies a name as lizardfolk; never omit it
- Names are likely personal + clan combined (a single compound word vs. two-word forms)
- The pool contains only subrace names, not personal names; infer that personal names follow the same Ix- + consonant-complex + terminal pattern

## Worked examples (derived, not copied)
- **Ixnokrath** — Ix- + -no- + -krath (cluster terminal); a personal or clan name
- **Ixvelsuun** — Ix- + -vels- + -uun (elongated terminal); a settlement or tribal name
