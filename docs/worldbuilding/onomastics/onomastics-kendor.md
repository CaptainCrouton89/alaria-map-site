# Kendor Onomastics

> **sparse sample — extrapolate conservatively**

**Author anchor:** Soft, flowing — liquid consonants, open vowels, -ndor/-lian/-dor terminals  
**Source pool (frozen map names — do not copy verbatim):** Cendoriln, Kendrel, Pelaendor, Pelarian

## Sound palette
- Consonants that recur: liquids l/r/n, voiced fricative nd/el, soft d/k/c as openers; no hard stops; no clusters
- Vowel behavior: open mid vowels (e, a, i); flowing vowel-consonant alternation (Pelaendor = pel-a-en-dor); no compressed vowels
- Forbidden / off-style: hard guttural stops (k/g at close), consonant stacks, apostrophes, sibilant clusters — anything that breaks the liquid flow

## Morphemes & affixes
- The race name itself (Kendor) supplies the stem pattern: Ken-/Cen- opener + -dor/-doriln terminal
- Suffixes: **-dor** (Pelaendor, shared with race name), **-rel** (Kendrel), **-rian** (Pelarian), **-oriln** (Cendoriln = extended -dor variant)
- Prefixes: Pel- (Pelaendor, Pelarian), Cen-/Ken- (race-name family)
- Compounding: two soft roots joined with a vowel bridge (Pela-en-dor = three-beat compound)

## Naming customs
- The -dor terminal is the strongest identity signal (the race name ends in it; Pelaendor carries it); use it for formal or place names
- -rel and -rian are personal-name terminals (shorter, intimate)
- Names feel graceful and unhurried — 3–4 syllables is typical; monosyllables would read as Koren

## Worked examples (derived, not copied)
- **Velaendor** — place/formal: Vel- + -a- + -endor (full -dor compound); a settlement or ancestral name
- **Narelian** — personal: Nar- + -el- + -ian (personal terminal blend); a given name
