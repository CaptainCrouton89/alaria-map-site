# Halfling Onomastics

**Author anchor:** British/medieval English with light Germanic drift  
**Source pool (frozen map names — do not copy verbatim):** Hookling, Whiteling, Wispen, Windorf, Belenstrope, Grendenheim, Sestros, Morelous, Shailin, Dengar, Fieri, Chimea, Phirexes, Erasnus

## Sound palette
- Consonants that recur: w/h/l/n/s/r — soft and approachable; no guttural clusters, no uvulars
- Vowel behavior: short vowels in given names (Wispen, Dengar); open vowels in place names (Sestros, Morelous); Germanic branch allows ei/en endings (Grendenheim)
- Forbidden / off-style: hard k/g/x consonant clusters, apostrophe glottals, long consonant stacks — anything that reads orc or harsh-elf

## Morphemes & affixes
- Suffixes (English/personal): -ling (diminutive, affectionate: Hookling, Whiteling), -pen/-wen (given-name terminal: Wispen), -in (soft terminal: Shailin)
- Suffixes (Germanic/place): -heim (settlement: Grendenheim), -dorf/-dorf > -dorf (hamlet: Windorf)
- Suffixes (Romance/place): -os/-ous (civic: Sestros, Morelous), -ea (light: Chimea), -us (Latinized: Erasnus), -es (Latinized variant: Phirexes)
- Prefixes: Belen- / Bel- (bright/fair cognate), Gren- / Gren- (green/verdant), Win-/Wis- (knowledge/wind)
- Compounding: English-style descriptor + noun (White + -ling, Hook + -ling); keeps roots recognizable

## Naming customs
- **Personal names** favor -ling and -pen/-wen forms; warm and diminutive in feel
- **Settlement names** split by culture: rustic halfling villages use Germanic -heim/-dorf; coastal/trade halfling towns use Romance -os/-us
- **Clan/family names** often use a descriptive root + -ling; -strope is a rare compound-surname form attested only in Belenstrope — do not treat it as productive
- Regional divergence: halflings near Germanic human territories drift toward -heim/-dorf forms; halflings in trade cities adopt Romance -os/-us; purely halfling heartlands stay English (-ling, -pen)

## Worked examples (derived, not copied)
- **Brumbling** — personal name: Brum- (warm/soft English root) + -bling (variant -ling); a given name or nickname
- **Fernheim** — settlement: Fern- (fern/green descriptor) + -heim; a Germanic-style village name
- **Mirelon** — settlement: Mire- (marshy, English) + -lon (Romance softening of -on); a trade-town name
