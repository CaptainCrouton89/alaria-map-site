# Gnome Onomastics

**Author anchor:** Eclectic — high-syllable density, exotic consonants, English-descriptive coexist  
**Source pool (frozen map names — do not copy verbatim):** Tyberoskos, Nykotheryx Amberylika, Qipi, Ohblex, Foxborne, Techgnomes, Ansari, Bloodlings, Ezuri, Goshwen, Neka, Shyoka Saseidi — place names: Nekanzi Jungle, Blood Mountains, Cherry Blossom River Valley, Pelican Isles, Jungles of Titania, Gemstone Peaks

## Sound palette
- Consonants that recur: exotic stops (Qipi = q/k; Ohblex = bx cluster); high-frequency y/th/ryx; English descriptive branch uses plain English consonants
- Vowel behavior: two distinct modes — (a) long compound vowel chains (Amberylika = am-ber-y-li-ka, 5 syllables); (b) ultra-short single-syllable (Qipi, Neka = 2 syllables maximum)
- Forbidden / off-style: dwarven umlauts (ø/ü), orc guttural apostrophes, elven flowing -weir/-wir; gnome exotic ≠ elven exotic — avoid diphthong runs without consonant interruption

## Morphemes & affixes
- Prefixes (high-syllable): Tybero-/Nykot- (multi-consonant compound opener), Amber- (English-exotic blend)
- Prefixes (English-descriptive): Fox- / Tech- / Blood- / Gem- / Cherry- / Pelican- (literal English nouns)
- Suffixes (high-syllable): -oskos/-theryx/-ylika (extended terminals; length signals importance/age of name)
- Suffixes (short exotic): -pi / -lex / -wen (abrupt consonant terminal)
- Compounding: English-descriptive branch uses plain noun + noun or adjective + noun (Foxborne, Techgnomes, Gemstone Peaks, Blood Mountains); high-syllable branch chains 3+ syllable roots

## Naming customs
- Gnome onomastics deliberately refuses a single style — this is a feature, not an error
- **High-syllable names** (Tyberoskos, Nykotheryx Amberylika) signal ancient lineage, arcane profession, or formal context; double-word forms (Shyoka Saseidi) are family + given or title + name
- **Short exotic names** (Qipi, Ohblex, Neka) are informal, personal, or streetname-style
- **English-descriptive names** (Foxborne, Techgnomes) indicate a gnome enclave identified by its craft or geography; outsider-coined labels frequently stick
- Any gnome character or place can use any branch — the mix-within-culture is intentional

## Worked examples (derived, not copied)
- **Tykoryx** — high-syllable: Tyko- (Ty- onset from Tyberoskos + -ko- interior) + -ryx (-ryx terminal from Nykotheryx); a personal name or arcane title
- **Gemborne** — English-descriptive: Gem- (cf. Gemstone Peaks) + -borne (cf. Foxborne); a gnome enclave settlement
- **Oblyx** — short exotic: Ob- (cf. Ohblex opener) + -lyx (compressed exotic terminal); a personal name
