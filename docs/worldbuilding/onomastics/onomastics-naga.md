# Naga Onomastics

> **sparse sample — extrapolate conservatively**

**Author anchor:** Naga- prefix on formal/sacred names; soft multi-syllable stems  
**Source pool (frozen map names — do not copy verbatim):** Megelren, Nagakani, Nagashi

## Sound palette
- Consonants that recur: soft nasals (n/m), glides (g/k), liquids (r/l); no hard clusters; sibilants (sh/s) in terminal position
- Vowel behavior: open vowels (a, e, i); 3–4 syllables; vowel-prominent flow (Nagakani = na-ga-ka-ni, all open syllables)
- Forbidden / off-style: hard consonant clusters, apostrophe glottals, -obbi/-oxi orc suffixes, Ix- prefix (lizardfolk)

## Morphemes & affixes
- **Prefix (formal/sacred): Naga-** — two of three samples use Naga- as a prefix; treat it as the formal/sacred name register; informal or personal names may omit it (Megelren is prefix-free)
- Stems after Naga-: -kani (kani = open syllable stem), -shi (compressed terminal)
- Prefix-free form (Megelren): Meg- + -elren (soft compound); suggests informal or personal register
- Terminals: -ni/-shi/-ren — all soft; no hard stops

## Naming customs
- **Formal/sacred register:** Naga- prefix marks a name as significant (deity-name, sacred place, honored ancestor); always use for institutions, sacred sites, named divinities
- **Personal/informal register:** prefix-free; soft multi-syllable form (Megelren model)
- Two-register distinction: Naga-X for high-register, bare stem for everyday — do not use Naga- for an ordinary individual's name

## Worked examples (derived, not copied)
- **Nagasheni** — formal/sacred: Naga- + -sh- (from Nagashi) + -eni (-ni terminal from Nagakani); a sacred site or deity-name
- **Selenar** — informal personal: Sel- + -enar (soft liquid terminal); an everyday personal name
