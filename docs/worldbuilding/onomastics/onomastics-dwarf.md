# Dwarf Onomastics

**Author anchor:** Norse-flavored with Latin-influenced variants  
**Source pool (frozen map names — do not copy verbatim):** Strømgodden, Trømgar, Gondurak, Dnykuul, Gruynmar, Aureum, Verucan, Sennos, Movasi, Embly­dium, Einwissel, Fengruk, Hestrube, Nydor, Sennites, Yorinthian, Zwaeron

## Sound palette
- Consonants that recur: hard stops k/g/d/r, nasal clusters (mn, nd), fricatives (gr, str, zw), Nordic ø umlaut
- Vowel behavior: short vowels dominate; ø is umlaut-only, not general-purpose; Latin branch uses open ae/um endings
- Forbidden / off-style: soft glides (wh-, -ly), flowing diphthongs (ei, ou) outside compound boundaries, sibilant openings that read as elven

## Morphemes & affixes
- Prefixes: Strøm- (river/current cognate), Gr- / Gru- (growth/weight), Dn-/Dny- (depth, underground)
- Suffixes: -ak/-uul (place/clan terminals: Gondurak, Dnykuul), -mar/-gar (height/peak: Trømgar, Gruynmar), -um/-eum (Latin-branch prestige: Aureum, Emblydium), -os (civic Latin: Sennos)
- Compounding: Norse branch joins a descriptor root + -ak/-mar; Latin branch uses single-stem + -um/-an

## Naming customs
- **Clan names** signal lineage style: Norse-clan names end in -gar/-mar/-gar or use umlaut (Strøm-); Latin-clan names carry -eum/-an/-ites (Aureum, Sennites, Verucan)
- **Place names** follow the same split: Norse settlements use umlaut roots + -ak/-uul; civic/trade hubs lean Latin (-os, -ium)
- **Personal names** (thin pool) follow clan style — a member of Strøm­godden takes Nordic consonant clusters; one of Aureum takes open-vowel Latin forms
- Regional divergence: dwarves nearer the surface adopt more Latin smoothing; deep-delve clans stay purely Norse-harsh

## Worked examples (derived, not copied)
- **Grømnhak** — Norse branch: Grøm- (Gr- prefix from Gruynmar + ø umlaut from Trømgar) + -hak (hard-stop variant of -ak terminal, cf. Gondurak); a deep-clan name
- **Vorekum** — Latin branch: Vor- (labial consonant) + -ekum (-eum variant); a trade-hall name
- **Drazuul** — harsh middle ground: Dr- cluster + -azuul (-uul final with inserted vowel); a mine complex
