# Elf Onomastics

**Author anchor:** Tolkien-elf (soft branch) with a harsh-divergent regional branch  
**Source pool (frozen map names — do not copy verbatim):** Amverela, Eloweir, Xicrein, Enuau Kemjallb, Istora, Myorna, Caerene, Spindral, Teflin, Ythari — harsh branch: Kryaaji, Kuzagt, Sivakr, Rakiten, Vyko, Xalahir

## Sound palette

### Soft branch (mainstream)
- Consonants that recur: l/r/v/n/w, light aspirates; no guttural clusters
- Vowel behavior: open diphthongs and triphthongs welcome (Eloweir = el-o-weir); e/i/a dominant; final -a/-ra common
- Forbidden / off-style: hard stops k/g at word-open (reads harsh), double consonant clusters, umlaut

### Harsh branch (regional divergent)
- Consonants that recur: ky/kz/sk initial clusters, voiced j, tight kr/str; harder than the soft branch but shorter words
- Vowel behavior: compressed, often single short vowel between consonant clusters (Kryaaji, Sivakr)
- Forbidden / off-style: long flowing diphthongs from the soft branch (Eloweir-style vowel runs)

## Morphemes & affixes
- Prefixes (soft): El-/Elo- (light cognate), Am-/Amv- (grace), Ist- (star), My- (shadow/silver)
- Prefixes (harsh): Kry-/Kuz-/Siv-/Xal- — no shared meaning; phoneme-cluster openers
- Suffixes (soft): -wir/-weir (flow), -rela/-ela (grace/movement), -ra (place/people), -rin/-rein (radiant)
- Suffixes (harsh): -aji/-akt/-akr (terminal consonant or compressed vowel+stop)
- Compounding (soft): two roots joined by a vowel bridge (Enuau Kemjallb = two-word compound); the space is legitimate in elven compound names

## Naming customs
- **Subrace identity** determines branch: high/wood/star elves use the soft branch; dark elves and isolated harshened clans use the harsh branch
- **Personal names** mirror the branch strictly — mixing signals mixed-blood or exile
- **Place names** follow the inhabitant branch; a soft-elf settlement reads like Istora or Myorna; a harsh-elf stronghold reads like Kuzagt
- **Double-word names** (Enuau Kemjallb) are formal/ancestral; single-word names are everyday
- Regional divergence: the harsh branch is geographically concentrated; do not scatter harsh-elf names in regions dominated by soft elves

## Worked examples (derived, not copied)
- **Velorin** — soft branch: Vel- (light labial) + -o- bridge + -rin (radiant terminal); a personal name
- **Aelvara** — soft branch: Ael- (ae diphthong opener) + -vara (-vara = soft compound); a settlement name
- **Xuvakr** — harsh branch: Xu- (X onset from Xalahir) + -vakr (-akr terminal from Sivakr); a stronghold or clan name
